- Excel
- SQL
- Power BI

I am a 4th-semester Computer Science student looking for my first internship, trainee position or junior role in IT, specifically in data analysis.

I want to build projects that I can publish on GitHub and LinkedIn. The projects should be practical and should help me genuinely learn the tools, not just follow a course passively.

## Project

Project name:

Online Retail Sales Analysis

Dataset:

Online Retail dataset from UCI Machine Learning Repository.

The dataset contains transactional data from an online retail store, including invoice numbers, stock codes, product descriptions, quantities, invoice dates, unit prices, customer IDs and countries.
